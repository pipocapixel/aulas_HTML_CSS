ID e CLASS

ID

o ID fundiona como um CPF. Assim como só poddo ter um numero de CPF cada elemento so pode ter um valor de ID. O CPF é unico , ou seja, ninguem pode ter um CPF igual ao meu. Isso significa que o ID tambem e uncio , entao nao e possivel repetir o mesmo valor duas vezes.

CLASS

O CLASS funciona de maneira oposta ao ID. Diferente do ID em que você só pode ter um valor , no CLASS voce pode ter varios valores . Tambem diferente do Id em que nao se pode repetir um valor , no CLASS você pode utilizat o mesmo valor para varios elementos
